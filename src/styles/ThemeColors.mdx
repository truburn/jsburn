# Theme Color Scheme

All of the following colors are part of the theme within the `theme.color` property. For example, `theme.color.primary` to use the primary color. The colors are defined in the `@constants/color.json` file for each color variant.

  <ColorItem
    title="Base Colors"
    subtitle="Colors for common elements"
    colors={{
      bg: theme.color.bg,
      fg: theme.color.fg,
      muted: theme.color.muted
    }}
  />
  <ColorItem
    title="Brand Colors"
